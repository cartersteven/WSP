What I hope to learn

I hope to learn how to create a good-looking header. Most of my work with headers to date has been very plain by comparison with many of today’s trends. I hope to modernize my understanding of headers and their design.

How this relates to what I have/will learn in WDD

This will help my future in WDD because I will be able to make a header that looks like it belongs in this decade, and not one from the early 2000's. I also believe it will compliment what I have learned about developing headers.

How would I build this?

  • Create a header that spans the entire width of the top of the page, creating the component
  • Set an img as the logo in the middle and center it using CSS
  • Use spans to create the links, and use CSS to add background images and styling to them.
