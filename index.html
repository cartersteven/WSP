Prominent Sub-Navigation

What I hope to learn

Most subnavigation is vary nondescriptive of what it will take you to. I hope to learn to make a subnavigation that helps users to know exactly what they are selecting. The use of images and a short paragraph will accomplish this feat.

How this relates to what I have/will learn in WDD

This component will help me learn to separate my website's purposes into easily found and utilized sections. It helps create a better user experience and facilitates faster user retention of the purpose and function of the website.

How would I build this?

  • Create a section that defines the shape of the component.
  • Create a ul which contains all of the subnavigation sections within each li.
  • Fill each li with a h2 at the top, followed by an img that describes the function or use of that part of the site, a p that describes in detail the function of that section, and an a which contains the link to the part of the site in question.
